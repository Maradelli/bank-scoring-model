# Credit Default Risk Prediction
PET PROJECT: Development the model for evaluation of the debtor to decrease the NPL.

Цель данного проекта: разработка модели машинного обучения для оценки кредитоспособности заёмщиков с целью снижения доли невозвратных кредитов NPL.

# Данные:
Название датасета: Credit Card customer

Ссылка (Kaggle): [Датасет](https://www.kaggle.com/datasets/sakshigoyal7/credit-card-customers)

# Методология:
1) EDA (корреляционный анализ, проверка на мультиколлениарность)
2) Feature Engineering (создание новых фичей, WoE-преобразование)
3) Выбор моделей (Logistic Regression в качестве базы, XGBoost для большей точности)
